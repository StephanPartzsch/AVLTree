## Installation via CocoaPods

- Install CocoaPods. See [http://cocoapods.org](http://cocoapods.org)
- Add the AVLTree reference to your Podfile:
``
    platform :ios
    	pod 'AVLTree'
    end
``
- Run `pod install` or `pod update` from the command line
